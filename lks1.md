# lks1.md — 10년의 약속 (원드라이브 동기화 전용 메모리)
> 이광수 전용 · 새 세션 시작 시 반드시 가장 먼저 읽을 것

---

## 🏗️ 확정된 기술 스택
- **프론트엔드**: React 19 + Vite + TailwindCSS + react-router-dom
- **백엔드/DB**: Supabase (PostgreSQL + Auth + Storage)
- **알림/문자**: Solapi (SMS 알림)
- **지도/주소**: 카카오 우편번호 서비스 (`react-daum-postcode`)
- **버전 관리**: 원드라이브(OneDrive) 자동 동기화 방식 사용 (Git 미사용)

---

## 🚨 절대 변경 불가 규칙
1. **`fee_settings` Hard-coding 절대 금지** — 모든 수수료/금액은 관리자 환경설정에서 입력
2. **물리적 DELETE 절대 금지** — 삭제 시 `is_deleted` 상태값으로 관리
3. **수수료 소급 적용 금지**
4. **새 파일/폴더 생성 전 반드시 먼저 물어보고 논의할 것**
5. **어려운 기술 용어 대신 알기 쉽게 설명할 것**

## 📌 직전 작업/진행 현황 (현재 상황)

1. **[완료] V1 런칭 전 필수 보안 자물쇠 적용**
   - 개발 및 테스트용 `/dev-login` 404 차단 완료.
   - DB에 RLS 보안 장치 활성화 및 기존 비밀번호 해싱 후 `auth.users` 로 복제 마이그레이션 적용.
   - 메인 홈페이지의 롤링 마퀴와 긴급 접수를 안전한 RPC 파이프로 대체.
   - 딜러/팀장 로그인 체계를 Supabase JWT 공식 Auth로 전환.

2. **[다음 할 일] 맞춤형 상품(커스텀 패키지) 제안 시스템 개발 (여기부터 개발 시작)**
