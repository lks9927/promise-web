## 배포 전 해야 할 것

### 1. Supabase Replication 확인 (직접)
Supabase Dashboard → Database → Replication

6개 테이블 토글이 켜져 있는지 확인:
- funeral_cases, settlements, partners, coupons, vendors, funeral_progress_reports

**안 켜져 있으면** SQL Editor에서 실행:
```sql
ALTER PUBLICATION supabase_realtime ADD TABLE funeral_cases, settlements, partners, coupons, vendors, funeral_progress_reports;
```

### 2. 로컬 테스트
```bash
npm run dev
```

테스트 방법:
1. 브라우저 탭 2개 열기 (관리자 + 딜러)
2. 딜러에서 접수 등록
3. 관리자 화면에 **새로고침 없이** 접수 표시 확인
4. 토스트 🚨 알림 확인

### 3. 배포
```bash
npx vercel --prod
```
