> 난이도: 🟢 쉬움 | 소요: 반나절 | 기능 변화: 없음

## 현재 문제
- `AdminDashboard.jsx` = **1,748줄** (112KB) 한 파일에 모든 탭이 들어있음
- 수정할 때 찾기 어렵고, 한 탭 고치려다 다른 탭 실수로 건드릴 위험

## 목표
```
src/pages/AdminDashboard.jsx (150줄 = 레이아웃 + 탭 전환만)
src/components/admin/tabs/
├── CaseTab.jsx          (접수 현황)
├── SettlementTab.jsx    (정산 관리 — 이미 분리됨: SettlementManager.jsx)
├── PartnerTab.jsx       (파트너 관리)
├── CouponPanel.jsx      (쿠폰 관리)
├── VendorPanel.jsx      (외주업체 승인)
├── MessageTab.jsx       (이미 분리됨: AdminMessageTab.jsx)
└── CommissionTab.jsx    (이미 분리됨: CommissionSettings.jsx)
```
